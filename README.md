# Hello World for App Engine

This is a sample "Hello World" application for the "Automating Deployments for Node Apps on GCP" tutorial that works on App Engine.

## Setup

```
npm install
```

## Running Locally

```
npm start
```

## Testing

```
npm test
```

## Deploying to App Engine

```
gcloud app deploy
```
